# Grama Bites 🌾

Grama Bites is a modern food ordering web application built with React.

This project is inspired by real-world food delivery platforms and is being developed as part of my React learning journey through the Namaste React course. While the course provides the learning path, Grama Bites has its own identity, branding, and will continue to evolve with additional features.

## 🚀 Tech Stack

- HTML5
- JavaScript (ES6+)
- React
- ReactDOM
- Tailwind CSS

## 🎯 Goals

- Learn React fundamentals
- Build reusable components
- Understand JSX and the Virtual DOM
- Implement routing
- Add search and filtering
- Build a shopping cart
- Optimize performance
- Deploy the application
